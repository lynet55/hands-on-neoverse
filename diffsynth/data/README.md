# MANO Models

This directory stores required MANO model files, but due to licensing
restrictions **they cannot be included in this repository**.

To use MANO-based hand reconstruction:

1. Register at https://mano.is.tue.mpg.de/  
2. Download `mano_v1_2.zip`
3. Extract:
       MANO_LEFT.pkl
       MANO_RIGHT.pkl
4. Place them in:

       diffsynth/data/mano/models

This folder is ignored by Git.
